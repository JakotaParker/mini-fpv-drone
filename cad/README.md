# CAD Directory

Store exported design files by revision. Onshape remains the cloud CAD source of truth, while GitHub stores reviewable exports and revision documentation.

## Recommended Revision Structure

- `frame-v1/`
- `frame-v2/`
- `camera-mount-v1/`
- `antenna-mount-v1/`

Each revision folder should contain:

- Onshape link
- STL export
- STEP export
- Screenshots
- Critical dimensions
- Known issues
- Test status
