# Firmware

Use this directory for Betaflight configuration records.

## Save

- Flight-controller target and firmware version
- Date flashed
- Port configuration
- Receiver protocol
- Modes
- Motor protocol
- Motor order and direction
- Failsafe settings
- OSD settings
- VTX table
- CLI backups

## Suggested Filenames

- `betaflight-backup-before-setup-YYYY-MM-DD.txt`
- `betaflight-backup-first-hover-YYYY-MM-DD.txt`
- `betaflight-settings.md`
