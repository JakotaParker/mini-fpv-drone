# Manufacturing

This directory records how each physical revision was produced.

Do not rely on memory for slicer settings. Every print test should record:

- Printer
- Material
- Nozzle size
- Nozzle temperature
- Bed temperature
- Layer height
- Line width
- Wall count
- Top and bottom layers
- Infill
- Print speed
- Cooling
- Supports
- Build-plate adhesion
- Estimated and actual print time
- Result
