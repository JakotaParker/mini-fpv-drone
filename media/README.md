# Media

Store compressed, labeled documentation images here.

## Naming Examples

- `2026-07-22-frame-v1-onshape-isometric.png`
- `2026-07-25-frame-v1-print-failure.jpg`
- `2026-07-27-frame-v1-motor-fit.jpg`

Avoid committing large raw videos directly to GitHub. Upload large videos elsewhere and add a link in the relevant build-log or test file.
