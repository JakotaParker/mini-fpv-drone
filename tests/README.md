# Tests

Create a separate file for every meaningful test.

## Suggested Test Files

- `frame-v1-cad-review.md`
- `frame-v1-print-test-YYYY-MM-DD.md`
- `frame-v1-dimensional-inspection.md`
- `frame-v1-component-fit.md`
- `electronics-continuity-test.md`
- `smoke-stopper-test.md`
- `motor-direction-test.md`
- `first-hover-test.md`

Every test record should include:

- Date
- Revision
- Objective
- Setup
- Procedure
- Expected result
- Actual result
- Pass/fail decision
- Photos or evidence
- Problems
- Corrective action
